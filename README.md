# Felipe-Station 🚀

Welcome to **Felipe-Station**, my personal hub where you can access all my projects, social media, and portfolio in one place. A clean, modern, and mobile-friendly Linktree-style page designed to showcase my work.

---
## 🖼 Preview

![Felipe-Station Preview](/preview.png)

## 🎯 Features

- Centralized access to all my **projects and social profiles**
- Fully **responsive and mobile-friendly**
- Stylish **glass-effect buttons** with smooth hover animations
- Direct email link compatible with any browser
- Easy to update with new projects or links

---

## 🛠 Technologies Used

- **HTML5**  
- **CSS3**  
- **Boxicons** (for social media icons)  
- **Flexbox** for responsive layout
